Sorting Algorithms Visualised


This website is a small student project to help visualise several sorting algorithms. There are six different algorithms on display. These are:

  • Simple Algorithms

  • Insertion Sort
  • Bubble Sort
  • Shell Sort
  • Selection Sort

  • Efficient Algorithms

  • Quick Sort
  • Merge Sort
  • Heap Sort

Each alogrithm gives you the option to choose a number of bars to sort, as well as a speed bar, to increase or decrease the speed of the sorting.

The program can also be stopped at any time using the stop button. Select one of the algorithms in the header above to begin




NOTE: Merge Sort, and Heap Sort, are not fully completed. They may be completed later on but at present do not have full functionality
